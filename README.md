# CSS Crash Course

## Overview

This repository contains my personal CSS crash course notes. I created this project while learning the fundamentals of CSS and how styling works in modern web development.

The purpose of this project is to keep examples, notes, and quick references that I can revisit when building future websites and applications.

## Project Structure

The repository includes practice files and examples that demonstrate different CSS concepts.

Example topics covered include:

* CSS selectors
* Styling text and colors
* Box model
* Layout techniques
* Positioning
* Flexbox
* Grid
* Responsive design basics

As I continue learning, additional examples and notes will be added to this project.

## Purpose

This project helps me:

* Practice writing CSS from scratch
* Understand how styling affects webpage layout
* Build a personal reference for commonly used CSS properties
* Improve my workflow using Git and GitHub
